# Image API (image-api)

Image API was a digital business automation company specializing in the digitization, optimization, and management of business processes, forms, and content. The company has rebranded as **Docufree** and provides enterprise services including document scanning and imaging, digital mailroom automation, business process automation (RPA/AI), electronic signature, cloud document management, and outbound print and mail.

Despite the legacy brand name "Image API", the company does not publish a public developer API or API portal.
